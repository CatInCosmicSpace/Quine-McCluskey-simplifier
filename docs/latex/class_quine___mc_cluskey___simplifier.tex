\subsection{Подробное описание}
Метод Квайна-\/МакКласки. 

Данный класс позволяет минимизировать булеву функцию, заданную в виде наборов, на которых функция принимает значение 1, по методу Квайна-\/МакКласки.

Данные считываются из заданного входного файла, записываются в заданный выходной. Имена файлов следует подавать из командной строки.  Октябрь-\/ноябрь 2017 года.
