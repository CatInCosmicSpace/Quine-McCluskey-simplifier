Подробное описание

Метод Квайна-МакКласки.

Данный класс позволяет минимизировать булеву функцию, заданную в виде наборов, на которых функция принимает значение 1, по методу Квайна-МакКласки.

Данные считываются из заданного входного файла, записываются в заданный выходной. Имена файлов следует подавать из командной строки. Октябрь-ноябрь 2017 года.

◆ table_

std::vector<std::pair<std::vector<size_t>, bool> > Quine_McCluskey_Simplifier::table_
private

Таблица покрытия простыми импликантами
table представляет собой таблицу, которая хранит наборы и импликанты в виде:

Таблица покрытия
Набор 1Набор 2...Набор n
Импликант 110...1
Импликант 210...0
...............
Импликант m01...1

где 1 в ячейке ставится, если импликант j покрывает единицу функции на наборе i, 0 - иначе
Это вторая наиболее сложная структура, которая занимает \( O(n \cdot m) \) памяти, где \( n \) - количество единиц функции, \( m \) - количество импликант функции
