◆ state()

auto log_expr::exp_node::state ( ) const -> size_t

Нужен для определения стадии парсинга. В соответствии с стадией выкидываются исключения, означающие ошибки в процессе парсинга
Сложность \(O(1)\)

Аргументы

3если все поля заполнены
